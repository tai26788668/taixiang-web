# 關於我們 Section - 內容管理指南

## 📝 內容管理概述

本指南提供關於我們 section 的完整內容管理說明，包括基礎頁面和詳細頁面的內容更新、圖片管理和配置維護。

## 🎯 內容結構說明

### 基礎頁面內容
- **標題和副標題**: 主要標語和描述
- **公司描述**: 多段落的公司介紹，重點說明服務範圍
- **統計數據**: 數字化展示經驗年數、合作品牌數、銷售據點數
- **建築圖片**: 工廠外觀照片 (`building.jpg`)
- **核心特色**: 兩個主要特色亮點
  - 我們的服務 (代工 + 自有品牌)
  - 哪裡吃得到 (具體通路列表)

### 詳細頁面內容
- **頁面標題**: 詳細頁面的主標題
- **圖文卡片**: 兩張深度介紹卡片
  - 我們的服務：詳細說明代工和自有品牌服務
  - 哪裡吃得到：簡潔列出各大通路
- **導航元素**: 頁面切換控制項

## 📖 基礎頁面內容管理

### 修改標題和副標題

```typescript
about: {
  title: '關於我們',                    // 主標題
  subtitle: '傳承與創新的完美結合',      // 副標題
  // ...
}
```

**修改步驟**:
1. 開啟 `websiteConfig.ts`
2. 找到 `about.title` 和 `about.subtitle`
3. 修改文字內容
4. 儲存檔案並重新建置

### 更新公司描述

```typescript
description: [
  '第一段：公司服務範圍和專業能力介紹',
  '第二段：產品通路分布和市場覆蓋說明'

```

**內容建議**:
- 第一段：強調代工和自有品牌雙重服務能力
- 第二段：突出通路廣度和產品可及性

**新版範例**:
```typescript
description: [
  '泰鄉食品擁有三十年專業製餅經驗，提供完整的餅乾製造解決方案。我們不僅有自有品牌產品，更是眾多知名品牌的信賴代工夥伴。',
  '從傳統節慶餅乾到現代創新口味，我們的產品遍布全台各大通路，讓消費者在不同場合都能品嚐到泰鄉的用心製作。'
],
```

### 修改統計數據

```typescript
stats: [
  { value: '30+', label: '年製餅經驗' },
  { value: '50+', label: '合作品牌' },
  { value: '1000+', label: '銷售據點' }
```

**數據類型**:
- **經驗年數**: `30+` 突出專業資歷
- **合作規模**: `50+` 展現代工實力
- **通路覆蓋**: `1000+` 強調市場滲透
- **標籤**: 簡短描述性文字
- **建議**: 3-4 個關鍵數據，突出服務能力和市場地位

### 更新核心特色

```typescript
features: [
  {
    icon: '🏭',                           // 工廠圖示
    title: '我們的服務',                    // 服務特色標題
    description: '專業代工製造與自有品牌雙軌經營', // 服務描述
    gradient: 'from-blue-50 to-indigo-50'  // 藍色系背景
  },
  {
    icon: '🛒',                           // 購物圖示
    title: '哪裡吃得到',                    // 通路特色標題
    description: '台灣最大連鎖店 / 台灣最大美式賣場 / 甜點品牌 / 飛機餐 / 鄉間通路', // 通路列表
    gradient: 'from-green-50 to-emerald-50' // 綠色系背景
  }

```

**特色重點**:
- **我們的服務**: 強調代工和自有品牌雙重能力
- **哪裡吃得到**: 直接列出具體通路類型

**漸層色彩選項**:
- `from-amber-50 to-orange-50` (橙色系)
- `from-green-50 to-emerald-50` (綠色系)
- `from-blue-50 to-indigo-50` (藍色系)
- `from-purple-50 to-pink-50` (紫色系)
- `from-gray-50 to-slate-50` (灰色系)

### 管理圖文卡片

#### 卡片結構說明
```typescript
{
  id: 'heritage',                     // 唯一識別碼
  title: '傳統工藝傳承',              // 卡片標題
  description: '詳細內容描述...',      // 卡片內容
  imageSrc: '/images/heritage-craftsmanship.svg', // 圖片路徑
  imageAlt: '傳統製餅工藝師傅正在製作餅乾',        // 圖片描述
  order: 1                           // 顯示順序
}
```

#### 新增卡片
1. **準備圖片** (參考圖片管理章節)
2. **新增配置**:
   ```typescript
   cards: [
     // ... 現有卡片
     {
       id: 'new-card',
       title: '新卡片標題',
       description: '新卡片的詳細描述內容...',
       imageSrc: '/images/new-card-image.svg',
       imageAlt: '新卡片圖片的替代文字',
       order: 4
     }
   ]
   ```

#### 修改現有卡片
1. 找到要修改的卡片 (透過 `id` 識別)
2. 更新相應欄位
3. 儲存並重新建置

#### 刪除卡片
1. 從 `cards` 陣列中移除對應物件
2. 刪除相關圖片檔案 (可選)
3. 重新建置

#### 調整卡片順序
修改 `order` 欄位數值：
- `order: 1` - 第一個顯示
- `order: 2` - 第二個顯示
- `order: 3` - 第三個顯示

## 🖼️ 圖片資源管理

### 圖片規格要求

#### 技術規格
- **尺寸**: 1200x800px (3:2 比例)
- **格式**: SVG (推薦) / PNG / JPG / WebP
- **檔案大小**: < 500KB
- **色彩**: RGB 色彩空間

#### 內容建議
- **主題相關**: 與卡片內容高度相關
- **視覺品質**: 清晰、專業、美觀
- **風格一致**: 保持整體視覺風格統一
- **文化適宜**: 符合企業形象和文化背景

### 基礎頁面圖片管理

#### 建築圖片 (building.jpg)
- **位置**: `website/public/images/building.jpg`
- **用途**: 展示泰鄉食品工廠建築外觀
- **顯示位置**: 關於我們基礎頁面右側欄位上方
- **替代文字**: "泰鄉食品工廠建築外觀"

#### 更換建築圖片步驟
1. **準備新圖片** (建議尺寸: 1200x600px 或類似比例)
2. **替換檔案**: 將新圖片命名為 `building.jpg` 並放置到 `website/public/images/`
3. **重新建置**: `npm run build`
4. **檢查效果**: 確認圖片正確顯示

#### 檔案命名規範
```
heritage-craftsmanship.svg    ✅ 好的命名
natural-ingredients.svg       ✅ 好的命名
quality-innovation.svg        ✅ 好的命名

image1.jpg                    ❌ 避免使用
圖片-1.png                    ❌ 避免中文
my image.svg                  ❌ 避免空格
```

#### 目錄結構
```
website/public/images/
├── heritage-craftsmanship.svg
├── natural-ingredients.svg
├── quality-innovation.svg
└── [新增的圖片檔案]
```

#### 新增圖片步驟
1. **準備圖片檔案** (符合規格要求)
2. **放置到目錄**: `website/public/images/`
3. **更新配置**: 在 `websiteConfig.ts` 中設定路徑
4. **測試載入**: 確認圖片正常顯示

### 圖片最佳化

#### SVG 圖片最佳化
```bash
# 使用 SVGO 最佳化 SVG
npm install -g svgo
svgo input.svg -o output.svg
```

#### 點陣圖最佳化
- **線上工具**: TinyPNG, Squoosh
- **本地工具**: ImageOptim (Mac), FileOptimizer (Windows)
- **建置工具**: 整合到 Vite 建置流程

   - 測試各項功能正常
   - 檢查跨裝置顯示

## 📝 內容撰寫指南

### 標題撰寫原則
- **簡潔明確**: 10-15 字以內
- **突出重點**: 強調核心價值
- **易於理解**: 避免專業術語
- **吸引注意**: 具有感染力

### 描述內容撰寫
- **結構清晰**: 分段落組織內容
- **重點突出**: 使用關鍵詞強調
- **情感連結**: 建立與讀者的共鳴
- **行動導向**: 引導用戶進一步了解

### 文字長度建議
- **卡片標題**: 8-12 字
- **卡片描述**: 80-120 字
- **統計標籤**: 4-8 字
- **特色描述**: 15-25 字

### SEO 最佳化
- **關鍵詞**: 自然融入相關關鍵詞
- **語義化**: 使用語義化的標題結構
- **可讀性**: 保持內容易讀易懂
- **原創性**: 確保內容原創且有價值

## 🎨 視覺設計指南

### 色彩搭配
- **主色調**: 與品牌色彩保持一致
- **輔助色**: 使用漸層背景增加層次
- **對比度**: 確保文字可讀性
- **和諧性**: 整體色彩協調統一

### 排版設計
- **字體層級**: 標題、副標題、內文清晰區分
- **行距設定**: 保持適當的行距提升可讀性
- **對齊方式**: 統一的對齊方式
- **留白空間**: 適當的留白增加視覺舒適度

### 圖片選擇
- **主題相關**: 與內容高度相關
- **品質優良**: 高解析度、清晰度佳
- **風格統一**: 保持一致的視覺風格
- **情感表達**: 能夠傳達正確的情感訊息
