# 產品圖片目錄

此目錄包含網站產品展示所需的圖片檔案。

## 圖片規格建議
- 格式：JPG, PNG, WebP
- 尺寸：建議 200x200 像素以上
- 檔案大小：建議小於 500KB

## 產品圖片列表
- `mapong.jpg` - 麻粩產品圖片
- `cun-zao.jpg` - 寸棗產品圖片  
- `soda-crackers.jpg` - 蘇打餅乾產品圖片
- `soda-sandwich.jpg` - 蘇打夾心產品圖片
- `default-product.jpg` - 預設產品圖片（當其他圖片載入失敗時使用）

## 使用方式
圖片會在 Products 組件中自動載入，路徑為 `/images/products/[檔名]`
