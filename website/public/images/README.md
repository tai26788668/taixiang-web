# 關於我們詳細頁面圖片說明

## 📸 當前圖片狀態

本目錄包含關於我們詳細頁面所需的三張圖片，目前使用高品質 SVG 佔位符：

### 1. heritage-craftsmanship.svg
- **用途**: 傳統工藝傳承卡片
- **內容**: 展示傳統製餅工藝場景
- **建議替換**: 實際的師傅製作餅乾照片
- **尺寸建議**: 1200x800px (3:2 比例)

### 2. natural-ingredients.svg  
- **用途**: 嚴選天然食材卡片
- **內容**: 展示天然食材和農產品
- **建議替換**: 實際的食材照片或農場合作照片
- **尺寸建議**: 1200x800px (3:2 比例)

### 3. quality-innovation.svg
- **用途**: 創新與品質並重卡片  
- **內容**: 展示現代化生產設備和品質控制
- **建議替換**: 實際的生產線或品質檢測照片
- **尺寸建議**: 1200x800px (3:2 比例)

## 🔄 如何替換為實際照片

1. **準備照片**
   - 拍攝或取得高品質的實際照片
   - 建議尺寸：1200x800px (3:2 比例)
   - 格式：JPG 或 WebP (建議 JPG)
   - 品質：80-90%

2. **檔案命名**
   - 保持相同的檔案名稱，但改為 .jpg 副檔名
   - `heritage-craftsmanship.jpg`
   - `natural-ingredients.jpg` 
   - `quality-innovation.jpg`

3. **更新配置**
   - 編輯 `website/src/config/websiteConfig.ts`
   - 將 `.svg` 副檔名改為 `.jpg`
   - 更新 `imageAlt` 文字以符合實際照片內容

4. **圖片最佳化**
   - 使用 TinyPNG 或類似工具壓縮圖片
   - 確保檔案大小適中（建議 < 500KB）
   - 考慮提供 WebP 格式以提升效能

## 📋 圖片內容建議

### 傳統工藝傳承 (heritage-craftsmanship)
- 師傅正在製作餅乾的場景
- 傳統工具和製作台
- 手工製作的細節特寫
- 溫馨的工作環境

### 嚴選天然食材 (natural-ingredients)  
- 新鮮的製餅原料展示
- 麵粉、糖、油品等食材
- 與農民合作的場景
- 食材品質檢查過程

### 創新與品質並重 (quality-innovation)
- 現代化的生產設備
- 品質檢測儀器和流程
- 包裝和出貨場景
- 整潔的生產環境

## 🎯 無障礙考量

確保所有圖片都有：
- 描述性的 alt 文字
- 適當的對比度
- 清晰的視覺內容
- 支援螢幕閱讀器

## 📱 響應式設計

圖片將在不同裝置上以不同佈局顯示：
- **桌面**: 三欄網格 (1200px 寬度)
- **平板**: 兩欄網格 (768px 寬度)  
- **手機**: 單欄堆疊 (375px 寬度)

LazyImage 組件會自動處理載入和錯誤狀態。
