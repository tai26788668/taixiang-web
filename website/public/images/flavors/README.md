# 產品口味圖片目錄

此目錄包含產品口味頁面使用的圖片檔案。

## 圖片規格

- **格式**: JPG
- **尺寸**: 建議 400x300px 或更高解析度 (寬度比例)
- **品質**: 高品質，適合網頁顯示
- **命名**: 使用英文小寫，描述性檔名

## 圖片清單

| 檔名 | 對應產品 | 說明 |
|------|----------|------|
| `sesame.jpg` | 麻粩 | 芝麻相關圖片 |
| `peanut.jpg` | 花生粩 | 花生相關圖片 |
| `rice.jpg` | 米粩 | 米類相關圖片 |
| `scallion.jpg` | 青蔥蘇打餅 | 青蔥相關圖片 |
| `pepper.jpg` | 胡椒蘇打餅 | 胡椒相關圖片 |
| `special.jpg` | 芝麻蘇打餅 | 特調口味相關圖片 |

## 使用方式

圖片會在產品口味頁面中以 90% 寬度顯示在整個口味卡片區塊中，不再限制在圓形容器內。

## 備註

- 如果圖片載入失敗，會自動顯示產品名稱的第一個字作為備用
- 圖片會有 hover 效果 (放大 105%)
- 圖片使用 rounded-lg 樣式，具有圓角效果
- 圖片會自動調整高度以保持比例
